# Lab 1 - Intercepting Requests

## Goal
Intercept HTTP requests to Juice Shop using Burp Suite.

## Steps
1. Started Burp Suite
2. Configured Firefox proxy to 127.0.0.1:8080
3. Opened Juice Shop homepage
4. Captured request in Proxy tab
