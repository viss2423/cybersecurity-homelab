# Cybersecurity Homelab

A hands-on cybersecurity learning environment for practicing penetration testing, network security, and ethical hacking techniques.

## Structure

- `0-setup/` - Initial VM and network configuration
- `1-recon-enum/` - Reconnaissance and enumeration activities
- `2-initial-access/` - Initial access techniques and documentation
- `scripts/` - Automation scripts for common tasks

## Getting Started

1. Follow the setup guide in `0-setup/vm-network-setup.md`
2. Begin reconnaissance activities in `1-recon-enum/`
3. Document findings and techniques as you progress

## Disclaimer

This homelab is for educational purposes only. Only test on systems you own or have explicit permission to test.
